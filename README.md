# nam-phuong.fr

Site personnel créé avec [VitePress](https://vitepress.dev).

## Édition locale

Installer les dépendances :
```bash
bun install
```

Lancer le serveur de développement :
```bash
bun run vp:dev
```

Le site sera accessible à `http://localhost:5173`. Les modifications sont appliquées en temps réel.

## Publication

Après vos modifications locales, publier les changements :
```bash
git push
```

Les pages Markdown se trouvent dans le dossier `content/`.
